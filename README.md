# SauPiano
一个将你的键盘变为midi控制器，并且支持通过添加sf2音源文件夹来进行音色切换的表演软件（默认为Microsoft GS Wavetable Synth），同时支持播放演示.mid文件等功能 //  A performance software that turns your keyboard into a MIDI controller and allows you to switch sounds by adding an SF2 sound source folder (default: Microsoft GS Wavetable Synth); it also supports features such as playing .mid demo files. And more
